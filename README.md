# wechat_detection

image_demo.py展示了使用三种方法对文件夹内图片进行目标检测的过程

环境配置：
1. 创建环境
    python == 3.9
    pytorch == 1.12.1
    cuda == 11.6
2. 安装openmim，mmcv-full，mmdetection, opencv。参考https://github.com/open-mmlab
    mmcv-full == 1.6.1
    mmdet == 2.25.1
3. 运行image_demo.py测试
